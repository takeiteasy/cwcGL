glWindowCallbacks

Assign window event callbacks 

EXPORT void glWindowCallbacks(
    GLW_CALLBACKS void *userdata);
Parameters
KeyboardCallback

Callback for keyboard events

MouseButtonCallback

Callback for mouse button events

MouseMoveCallback

Callback for mouse movement event

MouseScrollCallback

Callback for mouse scroll event

ResizedCallback

Callback for window resize event

FocusCallback

Callback for window focus/blur events

ClosedCallback

Callback for window close event

userdata

Userdata stored by pp, passed to callback functions

Discussion

All callbacks are optional, pass NULL to skip callback. These can also be set individually, e.g. glWindowKeyboardCallback(cb)

GLflags

Window config flags 

typedef enum { 
    glResizable = 1 << 0, 
    glFullscreen = 1 << 1, 
    glFullscreenDesktop = 1 << 2, 
    glBorderless = 1 << 3, 
    glAlwaysOnTop = 1 << 4 
} GLflags;
Constants
glResizable

Resizable window flag

glFullscreen

Fullscreen window flag

glFullscreenDesktop

Window will resize to size of screen

glBorderless

Borderless window flag

glAlwaysOnTop

Floating window flag

Discussion

Flags can be shifted together with | operator

typedef enum { 
    KEY_MOD_SHIFT = 1 << 0, 
    KEY_MOD_CONTROL = 1 << 1, 
    KEY_MOD_ALT = 1 << 2, 
    KEY_MOD_SUPER = 1 << 3, 
    KEY_MOD_CAPS_LOCK = 1 << 4, 
    KEY_MOD_NUM_LOCK = 1 << 5 
} GLmod;
Constants

KEY_MOD_CONTROL

KEY_MOD_ALT

Alt key modifier

KEY_MOD_SUPER

Windows/Super key modifier

KEY_MOD_CAPS_LOCK

Caps lock key modifier

KEY_MOD_NUM_LOCK

Num lock key modifier

Discussion

Multiple modifiers will be shifted together so check with & operator
