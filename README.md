# cwcGL

_cwcGL_ is an OpenGL loader for C/C++ generated in Ruby. The actual dynamic library loader was taken from [glad](https://github.com/Dav1dde/glad) (also MIT).

This is just a fun side project.

> [!WARNING]
> Work in progress
